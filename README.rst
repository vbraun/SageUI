SageUI
======

A proposal for a graphical user interface to Sage/Trac/Git.

* Enhanced terminal for running Sage

* Sage trac browser / editor

* Interface to the Sage Git repository

Demo screencast: http://youtu.be/vsWo8lGRRlA
